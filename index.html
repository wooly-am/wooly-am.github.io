3d Paramatizer

A web-based synthesis environment

The user uploads a 3d model, and selects a line that will be the envelope. The program samples the highest point along the model, and constructs a periodic waveform that best fits the sample, avoiding aliasing issues. The waveform is then used in an oscillator to modulate a designated parameter.

DOODLESynth

A web-based synthesizer

10 voices, 4 "colours" or oscillators. The DOODLEsynth contains a grid where users can draw the outline of a waveform, that is fed into a buffer, and resampled for each note on-demand. If I were to do this project again, I would do a similar resynthesis as 3d Paramatizer, and avoid the issue of the buffer length having to round for certain frequencies.

MIDI Gesture Control

Midi controller using PD and mediapipe to interperate hand movement

his is a patch for PD Vanilla that implements MediaPipe hand landmarks, allowing users to to control PD patches, or use a virtual MIDI device to control DAWs, digital instruments and sequencers.This patch is not a full-on external, but includes the main image processing patch as well as other helpful patches for processing hand landmark positions. In addition to MediaPipe, I also used the PD external py4pd and the python opencv library.
